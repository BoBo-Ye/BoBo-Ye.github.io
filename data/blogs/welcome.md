## A quiet place for notes

This page is powered by a plain Markdown file under `data/blogs/`. Add a new `.md` file, list it in `data/blogs/index.json`, and the blog page will render it with the same visual rhythm as the rest of the site.

### Supported Markdown

- Headings from `#` to `######`
- Paragraphs and line wrapping
- Ordered and unordered lists
- Block quotes
- Links, images, inline code, bold text, and italic text
- Fenced code blocks

```js
const note = "Write the idea down while it is still warm.";
console.log(note);
```

> Good notes do not need to be loud. They just need a place to land.

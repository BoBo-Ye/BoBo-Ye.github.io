## CC Switch

CC Switch 用于 **统一管理 AI 编程工具配置切换**，适合在不同 API 服务（如：Claude Code）和模型配置之间快速切换。

- 支持各大 AI 厂商预设模板，快速添加常用供应商。
- 一键切换多个 API 供应商配置，减少反复改配置的麻烦。
- 配备用量查询、余额显示与端点测速功能。

🔗：[https://www.ccswitch.io/zh/](https://www.ccswitch.io/zh/)

## Clink

Clink 可以增强 Windows 里的命令行体验，尤其适用 `cmd`，它最明显的提升是**命令补全**、**历史搜索**和更舒服的输入体验，视觉上也有更为明显的**彩色高亮区分**。

- 在终端里快速搜索之前执行过的命令，避免重复输入长路径或长参数。
- 对 Git、目录路径、命令参数做更顺手的补全。
- 让 Windows 命令行更接近 Linux shell 的操作手感。

🔗：[https://github.com/chrisant996/clink/releases](https://github.com/chrisant996/clink/releases)

## Filezilla

FileZilla 是个 **FTP/SFTP 文件传输工具**，用于在本地电脑和服务器之间传文件。Windows UI 界面比较直观，左边是本地目录，右边是远程目录，拖拽就能完成上传和下载。

- 连接实验室服务器、云服务器，快速上传代码、数据和配置文件。
- 从服务器下载实验结果、日志、模型权重或生成的图片表格。
- 保存常用站点信息，减少每次手动输入主机地址、端口和账号的麻烦。

🔗：[https://filezilla-project.org/](https://filezilla-project.org/)

## Snipaste

Snipaste 是日常截图的小工具，它的核心是能把**截图临时钉在屏幕**上，同时**自带标注工具**，做对照、复现步骤时非常顺手。

- 读论文时，把公式、图表、流程图贴在屏幕一角，旁边继续看正文。
- 写代码时，把报错信息、接口说明或实验结果贴出来，减少来回切窗口。
- 做汇报或整理笔记时，快速截取局部内容，配合标注工具圈出重点。

🔗：[https://www.snipaste.com/](https://www.snipaste.com/)

## UV

UV 是 **Python 项目管理工具**，它把虚拟环境、依赖安装、脚本运行和锁文件管理集成在了一起，个人使用体验比conda要好。

- 新建一个轻量 Python 项目，并自动管理 `.venv`。
- 给已有项目添加依赖，保持 `pyproject.toml` 和锁文件同步。
- 在不同机器上用 `uv sync` 复现一致的依赖环境。

🔗：[https://docs.astral.sh/uv/](https://docs.astral.sh/uv/)

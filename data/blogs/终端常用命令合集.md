## 查看磁盘空间

查看各分区磁盘使用情况：

```
df -h
```

其中 `-h` 表示用 GB / MB 这种更容易读的单位显示。

查看当前目录下各文件/文件夹占用空间：

```
du -h --max-depth=1
```

按大小排序：

```
du -h --max-depth=1 | sort -h
```

查看某个目录占用空间，例如 `/data`：

```
du -sh /data
```



### 删除某个文件夹

删除空文件夹：

```
rmdir 文件夹名
```

删除非空文件夹：

```
rm -r 文件夹名
```

强制删除非空文件夹：

```
rm -rf 文件夹名
```

例如：

```
rm -rf /data/yehw2024/GridAgent/DATA
```
